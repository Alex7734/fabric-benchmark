About This Project

This project provides performance benchmarks for React Native applications, comparing the new Fabric architecture with the legacy architecture.

The aim is to identify performance improvements, bottlenecks, and areas that can be optimized for smoother rendering, better frame rates, and lower resource usage.

By presenting a series of test scenarios, we measure CPU, memory, and FPS usage under various conditions. The benchmarks help developers understand the impact of adopting Fabric and guide them towards building more performant React Native apps.
